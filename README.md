# Week 1: HTML Portfolio Website

## Project Overview
A personal portfolio website built using semantic HTML5 to showcase professional information, skills, and contact details.

## Setup Instructions
1. Clone this repository
2. Open `index.html` in any web browser
3. No additional dependencies required

## Code Structure
- `index.html` - Main HTML file with semantic structure
- `images/` - Directory for profile and gallery images
- `README.md` - Project documentation

## HTML Concepts Learned
- Semantic HTML tags (header, nav, main, section, footer)
- Form elements with validation attributes
- Internal navigation with anchor links
- Image embedding with alt attributes
- Proper document structure (DOCTYPE, head, body)

## Testing
Validated using W3C HTML Validator - All checks passed
Tested in Chrome, Firefox, and Edge browsers
